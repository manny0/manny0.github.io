用树莓派搭建轻量级家庭媒体中心-软件安装

开始之前

树莓派因其独特的硬件架构，虽然主频不高，但却可以硬解高清视频。非常适合用作低功率的高清解码方案，比如说家庭媒体中心。今天就来分享一下把小派折腾成家庭媒体中心的经历。XD

方案选择

了解到的技术方案主要有三种：

  • 使用别人做好的镜像，如OpenELEC、RASPBMC。镜像都可从树莓派主页下载。直接将下载的镜像使用Win32DiskImager等工具写入SD卡即可。
  • 自行下载XBMC源码，在小派上编译安装。介于小派的运算能力，此种方式非常耗时，据网上的说法大约为11-12小时。
  • 使用Rasbian里的终端安装别人已编译好的二进制包。由于小派的硬件是高度一致的，此种方法可行，也比较适合喜欢在终端里折腾的我。就是他了！XD

二进制包的安装

先附上XBMC官方安装说明文档。如文档中描述，我们来到这里查看安装编译好的二进制包的说明。安装过程还是有点小复杂的：

  • 添加第三方的源：在/etc/apt/sources.list.d/mene.list中添加
    deb http://archive.mene.za.net/raspbian wheezy contrib
  • 添加指纹标识：
    sudo apt-key adv —keyserver keyserver.ubuntu.com —recv-key 5243CDED
  • 更新软件包列表：
    sudo apt-get update
  • OK，安装XBMC！：
    sudo apt-get install xbmc
  • 确定使用XBMC的用户在以下用户组中：
    audio video input dialout plugdev tty
  • 若某用户组不存在，则创建。如input组：
    addgroup —system input

OK! 大功告成，在终端里键入XBMC试试吧！P.S.在SSH和VNC中是看不到界面的，还是找根HDMI线吧。

总结

至此，软件的安装是完成了，接下来的XBMC配置后续再补充吧。

在树莓派上安装Raspbian

今天的猪脚是树莓派（Raspberry Pi），一台信用卡大小的电脑。在X宝上只要200+RMB，可以运行linux，可以进行GPIO控制，还是很有玩头的。近日入了一台，准备用一个系列来记录折腾的脚步。下面开始折腾。

当然硬件到手后第一步就是安装OS了。

可以运行在树莓派上的发行版有很多，在这里。官方的NOOBS，可根据选择一键安装不同的发行版，但我是个喜欢折腾的，所以选择了RASPBIAN镜像来折腾。在Windows下安装的过程大致为：

  • 下载镜像及工具软件
  • 安装工具软件
  • 格式化SD卡
  • 拷贝镜像
  • 运行并配置RASPBIAN

RASPBIAN镜像及工具软件下载

从此处下载RASPBIAN镜像，也可搜索百度网盘资源（关键字：baiduyun raspbian），国内下载时速度快一些。

使用的工具软件为win32diskimager，项目主页。

工具软件安装

怎么装就不写了，玩这个的应该都会装的。：）

SD的格式化和镜像拷贝

使用系统自带的格式化工具将SD卡格式化，推荐使用4G以上的卡，现在卡也不贵，还可省却日后折腾多了以后空间不足的尴尬。还有另外一个选择是使用SDFormatter进行格式化，这是SD卡联盟推出的软件，应该靠谱一些。

格式化完成后重新插入SD卡，运行win32diskimager。点击蓝色的小文件夹按钮选择下载的镜像文件，下拉列表中选择SD卡的盘符。注意千万不要选错了盘符，因为写入后原来的数据会全部丢失。点击Write按钮，开始写入磁盘镜像。

运行RASPBIAN并配置

镜像写入完毕后，把卡安全弹出，插入到树莓派上，接上树莓派的电源。恭喜，应该可以看到linux启动的界面了。：）

配置界面直接跳过也是可以的，具体的配置选项请参见后续博文。

（改动中）（1）电子计算机组成——电子计算机基础知识

好久没动笔了，写的有点乱，正在调整，目前不建议阅读，各位海涵。

预热

今天学会了用hexo生成博客的方法，准备写点什么。想起早就答应老妹要教她点计算机基础知识，那就先写点这个吧，说不定可以假装下业余教材神马的。：）

说起电子计算机这个东西，相信现在是无人不知、无人不晓。满大街的智能手机，就是一种小型电子计算机，但其计算能力可是比著名的ENIAC（电子数值积分计算机（英语：Electronic Numerical Integrator And Computer））前辈高出许多倍。不得不感叹一下科技进步之神速。说句题外话，中学教材上说ENIAC是世界上第一台电子计算机，但疑为忽悠之词，证据在这，维基百科上说的比较准确——“世界上第一台通用电子计算机”。

电子计算机有个别称——电脑，实在是贴切至极之喻。为了便于理解，下面以人做比喻，形象说明“电脑”结构。

架构

正如人由身与神两部分组成，电子计算机由对应身的硬件和对应神的软件组成。

硬件组成（图表待补充）

从信息处理流程的角度，硬件组成可大致分为信息接收、储存、处理、发送四部分，各部分都可与人体某些器官相对应，信息的处理流程也与人体非常相似。

负责接收的主要包括麦克风、摄像头、键盘、鼠标、各种传感器等，正如人体上的耳朵、眼睛、皮肤等器官；负责储存的主要为磁盘、光驱、闪存、内存等，人体对应的为大脑中负责记忆储存的部分；负责信息处理的为大名鼎鼎的CPU和各种协处理器，正如人脑及脊髓中对各种信号进行反应的部分。

软件组成（软件分类、图表待补充）

硬件为信息的处理提供了各种通道和载体，而软件本身则是各种信息的载体。软件存储在存储器中，决定了硬件组成相同的计算机对同一信息的反应是否相同。对应于人，就如同身体健全的不同成年人，见到一个老太太倒在马路上了，有的人会去扶，有的则不然。正是因为存储在大脑中的“软件”不同。

总结

计算机组成分为硬件和软件两部分

  • 硬件
    • 输入设备
    • 输出设备
    • 存储器
    • 控制器
  • 软件
    • 操作系统
    • 应用软件
